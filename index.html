Jondo.com

Jondo is a global print on demand company.

As their first engineering hire, I built a bespoke ERP from the ground up, unifying order management, production, shipping, accounting, and analytics.

  • Orders originate in stores, transmitted via EDI
  • C# application routes image files into pre-production buckets
  • Application edits image, injecting a barcode for order tracking
  • C# application injects image into print queue via industrial raster software
  • Barcodes create shipping labels automatically

Accomplishments

  • Created company-wide ERP used to input, track, and manufacture orders
  • Innovated method to transmit images to industrial raster software with no API
  • Integrated directly with partners like Costco and UPS
  • Used C# and PHP to inject and read barcodes with printed products
  • Worked as sole developer, closely collaborating with stakeholders
  • Scaled to 1000s of orders per day

Skills Used

  • PHP
  • C#
  • mySQL
  • JavaScript
  • EDI
  • APIs
  • ERP
